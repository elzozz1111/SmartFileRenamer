# SmartFileRenamer 🚀

تطبيق احترافي لإعادة تسمية الملفات بذكاء مع واجهة رسومية حديثة.

## المميزات ✨

- 🤖 **محرك إعادة تسمية ذكي** باستخدام OpenAI API
- 🔐 **نظام تفعيل آمن** مع حماية HWID
- 🎨 **واجهة رسومية احترافية** مع ثيمات داكنة وفاتحة
- 💾 **قاعدة بيانات SQLite** مع دعم Undo/Redo
- ⚡ **عمليات خلفية** بدون تجميد الواجهة
- 📊 **إحصائيات وسجلات** مفصلة
- 🌐 **واجهة عربية** كاملة

## المتطلبات 📋

- Python 3.10+
- PyQt6
- OpenAI API Key (اختياري)

## التثبيت 🔧

### 1. استنساخ المستودع
```bash
git clone https://github.com/elzozz1111/SmartFileRenamer.git
cd SmartFileRenamer
```

### 2. تثبيت المتطلبات
```bash
pip install -r requirements.txt
```

### 3. (اختياري) إعداد OpenAI API
```bash
cp .env.example .env
# ثم عدّل .env وأضف مفتاح API
```

## التشغيل 🎯

```bash
python main.py
```

## الاستخدام 📖

### 1. التفعيل الأول
عند تشغيل البرنامج لأول مرة، ستظهر نافذة التفعيل:
- انسخ معرف الجهاز (HWID)
- أدخل مفتاح التفعيل الذي تلقيته
- اضغط "تفعيل"

### 2. إعادة تسمية الملفات
- اختر المجلد الذي يحتوي على الملفات
- اضغط "اقتراحات" للحصول على أسماء ذكية من AI
- اختر نمط التسمية
- اضغط "تطبيق التسمية"

### 3. نسخ الملفات
- اختر المجلد المصدر
- اضغط "نسخ"
- اختر مجلد الوجهة
